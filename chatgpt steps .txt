Step 4: Write your first parser

Create

parsers/linux_parser.py

Goal:

Read log.

Extract:

timestamp
username
IP
status

Start with:

import re


pattern = r'Failed password for (\w+) from ([\d\.]+)'


with open("samples/auth.log") as f:

    for line in f:

        match = re.search(pattern,line)

        if match:

            print(match.group(1))
            print(match.group(2))

Run.

python parsers/linux_parser.py

If it prints

root

10.0.0.5

Congratulations.

ThreatLens now exists.

Step 6: Build your first detector

    ip=e['ip']
